Social Scheduler Core

A unified library for scheduling and publishing content across social media platforms.

What is it?

Social Scheduler Core is an open‑source Python library that provides a clean, consistent interface for automating content across Twitter/X, Mastodon, LinkedIn, and more.

It’s designed as the engine behind content‑creator tools, marketing dashboards, and cross‑platform automation workflows.

Key Features

  • Twitter/X API v2 wrapper – post tweets, read timelines, retrieve user tweets
  • Modular design – easy to add new platform adapters
  • Environment‑based configuration – store API keys securely
  • Ready for PyPI – packaged as a standard Python library
  • MIT licensed – free for personal and commercial use
